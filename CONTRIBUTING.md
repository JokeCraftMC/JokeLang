# 🤝 Contributing
Thanks for helping translate JokeCraft!
## ✅ Before Submitting
- [ ] Only translated the **words**
- [ ] Kept `<tags>` unchanged
- [ ] Kept `{placeholders}` unchanged
- [ ] JSON is valid
## 🎯 Examples
### Good ✅
```json
"welcome": "<primary>Bienvenue, {player}!</primary>"
```
### Bad ❌
```json
"welcome": "<primaire>Bienvenue, {joueur}!</primaire>"
```
